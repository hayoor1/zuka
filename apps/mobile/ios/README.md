# iOS Native Project

This folder contains the iOS native project code for the Zuka mobile app.

## Generation

This folder is automatically generated by Expo when you run:
```bash
npx expo prebuild --platform ios
```

## Building

### Development
```bash
npm run ios
```

### Production
Open `ios/Zuka.xcworkspace` in Xcode and build for release.

## Requirements

- macOS
- Xcode 14+
- CocoaPods (`sudo gem install cocoapods`)

## Important Notes

- Do not manually edit files here unless necessary
- Changes should be made through Expo config plugins when possible
- This folder is gitignored by default
- Regenerate if you modify `app.json` configuration
